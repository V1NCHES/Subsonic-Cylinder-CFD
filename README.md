# Расчет дозвукового обтекания круглого цилиндра (нелинейное приближение)

## Постановка задачи
Рассматривается дозвуковое безвихревое обтекание круглого цилиндра. Задача сводится к решению нелинейного уравнения для потенциала скорости $\varphi$:
$$\frac{\partial}{\partial x} \left( \rho \frac{\partial \varphi}{\partial x} \right) + \frac{\partial}{\partial y} \left( \rho \frac{\partial \varphi}{\partial y} \right) = 0$$
где плотность $\rho$ определяется из условия изэнтропичности. 

Для решения используется конформное преобразование $\zeta = \ln z = \xi + i\eta$, переводящее внешность цилиндра в полуполосу.

### Требования:
1. Вывести уравнения и граничные условия в координатах $(\xi, \eta)$.
2. Реализовать ПНМ (Полностью Неявный Метод) и сравнить его с методом релаксации (SLOR).
3. Провести расчеты для различных чисел Маха ($M_\infty$).
4. Исследовать влияние параметров сетки и критическое число Маха.

## Реализация
В данном репозитории представлены две реализации численного решения:
1. **Standard_Solver**: Базовая реализация ПНМ с использованием SIP (Strongly Implicit Procedure).
2. **Accelerated_Solver**: Ускоренная версия ПНМ (Fast SIP), использующая несколько внутренних итераций на каждом шаге обновления плотности, что дает ускорение сходимости до 14 раз.

## Содержание репозитория
- `Report/`: Технический отчет (`document.pdf`), исходный код LaTeX и все графики результатов.
- `Materials/`: Справочные материалы, включая постановку задачи и статьи по методам SIP/PNM.
- `Standard_Solver/`: Код базового решателя.
- `Accelerated_Solver/`: Код оптимизированного решателя.
