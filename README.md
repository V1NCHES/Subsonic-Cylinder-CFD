# Исследование дозвукового обтекания круглого цилиндра

## Описание задачи
Данный проект посвящен численному моделированию дозвукового обтекания круглого цилиндра. Задача решается на основе метода конформных отображений, который позволяет перейти от физической плоскости к расчетной области с прямоугольной сеткой. 

Суть задачи заключается в нахождении распределения параметров потока (скорости, давления, числа Маха) вокруг цилиндра при различных числах Маха набегающего потока.

## Методы решения
В проекте реализованы и сравниваются несколько численных методов для решения разностных уравнений потенциала:
1.  **Полностью неявный метод (PNM) с факторизацией Булеева** (папка `PNM (Buleev)`).
2.  **Полностью неявный метод (PNM) с использованием сильно неявной процедуры (SIP)** (папка `PNM (SIP)`).
3.  **Метод последовательной верхней релаксации по линиям (SLOR)** (папка `SLOP`).

## Структура проекта
*   `PNM (Buleev)` — реализация и результаты расчета методом Булеева.
*   `PNM (SIP)` — реализация и результаты расчета методом SIP (Strongly Implicit Procedure).
*   `SLOP` — реализация и результаты расчета методом SLOR.
*   `tex` — исходные файлы LaTeX для оформления отчета, включая графики и теоретическое описание.
*   `task` — условия задачи, учебные материалы и вспомогательная литература.
