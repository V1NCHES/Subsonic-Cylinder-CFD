# План решения задачи обтекания цилиндра (синхронизированный с LaTeX)

Данный план полностью соответствует математической модели, описанной в файлах проекта (`3_dimensionless.tex` — `7_physics.tex`).

## 1. Постановка задачи и безразмерный вид (`3_dimensionless.tex`)
- **Обезразмеривание:** Переход к безразмерным переменным по масштабам $R$ (длина), $U_\infty$ (скорость), $\rho_\infty$ (плотность).
- **Уравнение для потенциала:** Решение уравнения неразрывности для безвихревого потока:
  $$\frac{\partial}{\partial x} \left( \rho \frac{\partial \varphi}{\partial x} \right) + \frac{\partial}{\partial y} \left( \rho \frac{\partial \varphi}{\partial y} \right) = 0$$
- **Связь плотности с потенциалом:** Использование изэнтропического закона и интеграла Бернулли:
  $$\rho = \left[ 1 + \frac{\gamma-1}{2} M_\infty^2 \left( 1 - q^2 \right) \right]^{\frac{1}{\gamma-1}}$$

## 2. Координатное преобразование (`4_transform.tex`)
- **Переход в $(\xi, \eta)$:** Использование полярно-логарифмических координат $x = e^\xi \cos \eta, y = e^\xi \sin \eta$.
- **Уравнение в новых координатах:**
  $$\frac{\partial}{\partial \xi} \left( \rho \frac{\partial \varphi}{\partial \xi} \right) + \frac{\partial}{\partial \eta} \left( \rho \frac{\partial \varphi}{\partial \eta} \right) = 0$$
- **Расчетная область:** $\xi \in [0, \xi_{max}]$, $\eta \in [0, 2\pi]$.

## 3. Численный метод SLOR (`6_slor.tex`)
- **Дискретизация:** Использование центральных разностей второго порядка на сетке $N_\xi \times N_\eta$.
- **Алгоритм SLOR:**
  1. Решение вдоль вертикальных линий $\eta = \text{const}$ (индекс $j$).
  2. Формирование трехдиагональной матрицы $A_i \phi_{i-1, j}^* + B_i \phi_{i, j}^* + C_i \phi_{i+1, j}^* = D_i$.
  3. Решение методом прогонки (`solve_tridiagonal`).
  4. Обновление: $\phi_{i, j}^{k+1} = \phi_{i, j}^k + \omega (\phi_{i, j}^* - \phi_{i, j}^k)$.
- **Граничные условия:**
  - **Цилиндр ($\xi=0$):** Условие Неймана $\phi_{-1,j} = \phi_{1,j}$ (реализуется через $C_0 = A_0 + C_0, A_0 = 0$).
  - **Бесконечность ($\xi=\xi_{max}$):** Условие Дирихле $\phi = e^\xi \cos \eta$.
  - **Оси ($\eta=0, 2\pi$):** Условия симметрии $\phi_{i,-1} = \phi_{i,1}$ и $\phi_{i,N_\eta+1} = \phi_{i,N_\eta-1}$. Применяется удвоение коэффициентов связи и вкладов в правую часть $D$ для крайних строк.

## 4. Расчет физических величин (`7_physics.tex`)
После сходимости итераций ($\epsilon < 10^{-5}$) вычисляются:
- **Квадрат скорости:** $q^2 = e^{-2\xi} [ \phi_\xi^2 + \phi_\eta^2 ]$.
- **Скорость звука:** $a^2 = 1 + \frac{\gamma-1}{2} M_\infty^2 (1 - q^2)$.
- **Локальное число Маха:** $M_{loc} = \frac{q M_\infty}{a}$.
- **Коэффициент давления:** $C_p = \frac{2}{\gamma M_\infty^2} (\rho^\gamma - 1)$.

## 5. Вычислительный эксперимент
- Расчет для чисел Маха $M \in \{0, 0.1, 0.2, 0.3, 0.4\}$.
- Построение графиков распределения $C_p(\eta)$ и $M_{loc}(\eta)$ на поверхности цилиндра.
- Сравнение с аналитическим решением для несжимаемого потока ($M=0$): $C_p = 1 - 4 \sin^2 \eta$.
